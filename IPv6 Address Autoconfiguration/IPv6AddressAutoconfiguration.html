ipv6 address autoconfig :

  • This command is used for Stateless Address Autoconfiguration (SLAAC). With SLAAC, a device can automatically configure an IPv6 address.
  • When a device is configured to use SLAAC for IPv6, it listens for Router Advertisements (RAs) sent by routers to determine the necessary IPv6 prefix and other network-specific information.Using the advertised information, it generates its own IPv6 address.
  • The key benefit is that there's no manual configuration of addresses and no need for a DHCP server.

ipv6 address 2001:aaaa:aaaa:aaaa::/64 eui-64:

This command configures an IPv6 address on the interface. The eui-64 keyword tells the router to use the EUI-64 process to generate the host portion (least significant 64 bits) of the IPv6 address.

EUI-64 stands for Extended Unique Identifier. It's a method of turning a 48-bit MAC address into a 64-bit address by inserting a specific 16-bit value (0xFFFE) into the middle of the MAC address.

When the command ipv6 address 2001:aaaa:aaaa:aaaa::/64 eui-64 is used, the router will generate the last 64 bits using its MAC address in the EUI-64 format.
